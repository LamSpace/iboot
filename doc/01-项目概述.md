# 01-项目概述

## 1. 项目简介

iBoot 是一个基于 Vue 3 和 Spring Boot 的通用管理系统，旨在为企业级应用提供统一的后台管理平台。项目采用前后端分离架构，前端使用 Vue 3 + TypeScript + Vite 构建，后端采用 Spring Boot 3.5.9 + JDK 25 技术栈。

**后端核心技术栈：**
- **框架**: Spring Boot 3.5.9、Spring Security 6.x、Spring Web 6.x、Spring Validation、Spring AOP、Spring Actuator
- **持久层**: MyBatis 3.5.x、MyBatis-Plus 3.5.x、PageHelper 2.1.0、Druid 1.2.24 连接池
- **安全认证**: Spring Security 6.x、JJWT 0.12.5、BCrypt 密码加密
- **缓存**: Redis 7.x、Caffeine 3.1.8 二级缓存架构
- **数据库迁移**: Flyway 10.20.1
- **对象存储**: MinIO 8.5.7
- **搜索引擎**: Elasticsearch 8.17.0
- **定时任务**: Quartz 2.3.x
- **API 文档**: Springdoc OpenAPI 2.8.13、Swagger UI
- **工具库**: Lombok 1.18.42、Apache Commons、Guava 33.4.0-jre、FastExcel 0.19.0
- **监控**: Micrometer、Prometheus、Grafana、Spring Boot Admin 3.4.5
- **消息推送**: SSE (Server-Sent Events)、Spring SseEmitter、Redis Pub/Sub（集群支持）

**前端核心技术栈：**
- **框架**: Vue 3.x、TypeScript ~5.4、Vite 5.x
- **状态管理**: Pinia 2.1.x
- **路由**: Vue Router 4.x
- **UI 组件**: Element Plus 2.x
- **HTTP**: Axios 1.6.x、@vueuse/core
- **可视化**: Three.js、ECharts
- **代码质量**: ESLint、Prettier、Stylelint

项目遵循 DDD（领域驱动设计）架构模式，采用分层架构设计，旨在提供一个可扩展、可维护的企业级解决方案。

### 1.1 项目目标

- 提供完整的系统管理能力，覆盖用户、角色、权限、菜单、数据字典、配置等基础能力
- 支持组织架构管理（部门、岗位、组织架构图）
- 提供个人中心、消息推送、安全管理（登录/操作/运行日志、敏感操作审计）
- 实现文件管理（上传、列表、预览、下载、删除、分类与权限控制、导入导出）
- 提供数据管理（备份与恢复、清理与归档、历史数据管理）
- 提供监控运维（在线用户、服务、Redis、SQL、性能监控、告警）
- 支持定时任务管理
- 提供统计分析（报表、使用分析、可视化图表、仪表板）

### 1.2 技术选型理由与优势

#### 后端技术选型

| 技术 | 选型理由 | 优势 |
|------|----------|------|
| Spring Boot 3.5.9 | 企业级后端框架，生态完善 | 自动配置简化开发、内嵌容器、生产就绪特性 |
| Spring Security 6.x | 强大的认证与访问控制框架 | 基于角色的访问控制（RBAC）、方法级安全注解、支持多种认证方式 |
| MyBatis/MyBatis-Plus | SQL 灵活可控，支持复杂查询 | MP 简化 CRUD、分页插件、逻辑删除、自动填充、条件构造器 |
| Druid 1.2.24 | 阿里巴巴连接池，监控强大 | 内置监控、WallFilter 防 SQL 注入、性能优秀 |
| Redis 7.x | 高性能缓存，支持多种数据结构 | 内存存储、Pub/Sub、集群支持、持久化 |
| Caffeine 3.1.8 | 本地缓存，与 Redis 组成二级缓存 | 高性能、基于 W-TinyLFU 算法 |
| Flyway 10.20.1 | 数据库版本管理 | SQL-based 迁移、支持回滚、与 Spring 集成好 |
| JWT 0.12.5 | 无状态认证，适合微服务架构 | 跨平台、支持自定义声明、滑动过期机制 |
| MinIO 8.5.7 | 开源对象存储，S3 兼容 | 高性能（183GB/s）、分布式部署、云原生友好 |
| Quartz 2.3.x | 功能强大的作业调度框架 | 支持集群、持久化、Cron 表达式灵活 |
| Elasticsearch 8.17.0 | 分布式搜索和分析引擎 | 实时搜索、聚合分析、ELK 日志体系 |

#### 前端技术选型

| 技术 | 选型理由 | 优势 |
|------|----------|------|
| Vue 3 + TypeScript | 现代化前端开发体验，类型安全 | 组合式 API、更好的类型推导、性能提升 |
| Element Plus | 丰富的 UI 组件库，开箱即用 | 组件齐全、文档完善、社区活跃 |
| Pinia | 轻量级状态管理 | 比 Vuex 更简洁、TypeScript 友好、1KB 轻量级 |
| Vue Router | 支持嵌套路由与权限控制 | 成熟稳定、功能完善 |
| Axios | 统一请求封装与拦截器 | 简洁易用、功能强大 |
| Vite | 快速构建与热更新 | 基于 ES Module 即时启动、多框架支持 |

### 1.3 业务价值与技术价值

**业务价值：**
- 通过统一的后台管理平台，降低企业数字化建设成本
- 提升管理效率与安全性
- 支持企业组织架构的灵活配置
- 完善的日志与审计体系，满足合规要求

**技术价值：**
- DDD 分层架构：接口层、应用层、领域层、基础设施层，职责清晰
- 依赖倒置原则：领域层不依赖外部框架，便于测试和维护
- 仓储模式：统一数据访问接口，支持多种持久化实现
- 领域事件：解耦业务逻辑，支持事件驱动架构
- 二级缓存架构：Caffeine + Redis，提升访问性能
- 无状态 JWT 认证：支持水平扩展和多端访问
- 完善的监控体系：Micrometer + Prometheus + Grafana + ELK
